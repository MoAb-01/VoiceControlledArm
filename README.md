## Website System
This is the frontend web application for controlling a prosthetic/robotic arm using voice commands. The site provides an intuitive user interface to connect with the arm via Bluetooth, process spoken input, and send commands in real time

## Tech stack 

Frontend: HTML, CSS, JavaScript

APIs/Libraries:

Web Speech API (for transcription)

Web Server API (for device communication)
